# 🐱 桌面宠物

一只会走路、眨眼、撒娇的小猫，住在你的桌面上。

## 使用

双击 `启动桌面宠物.bat` 即可。

或者命令行运行：
```
python desktop_pet.py
```

## 功能

| 操作 | 说明 |
|------|------|
| 🖱 拖拽 | 按住左键拖拽猫移动位置 |
| 🖱 右键 | 弹出菜单，切换状态/退出 |
| 🤖 自动 | 猫会自动走路、眨眼、发呆、睡觉、开心 |

## 状态

- **idle** 😐 正常状态，会轻微呼吸
- **blink** 😊 眨眼
- **walk** 🚶 在桌面上随机走动
- **sleep** 😴 睡觉（闭眼、特殊嘴型）
- **stretch** 🧘 伸懒腰
- **happy** 😆 开心（笑脸）

## 技术

- 纯 Python 标准库（tkinter），零依赖
- 程序化绘图，无需图片资源
- 透明背景，窗口置顶
- 跨平台（Windows / macOS / Linux）
