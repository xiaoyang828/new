## 一、Skill用途

本Skill基于**短视频爆款内容方法论**与**丁桂儿IP体系**，自动输出可直接用于AI视频生成（Runway/Pika/Kling/Sora等）的标准化分镜提示词。核心设计原则：

- **3秒钩子定生死** — 每个脚本前3秒必须完成注意力捕获
- **7-10秒节奏切换** — 镜头/信息点按科学频率排布，维持完播率
- **互动埋点贯穿** — 每个分镜设计互动钩子，引导评论/转发/收藏
- **AI生成兼容** — 提示词经结构化适配主流视频生成工具

## 二、输入解析

### 2.1 参数提取规则

用户输入格式：`<时长> <选题>`

```yaml
示例1: "30秒 儿童心肺复苏(CPR)保姆级教程"
-> duration: 30, topic: "儿童心肺复苏(CPR)"

示例2: "60秒 秋季腹泻怎么办"
-> duration: 60, topic: "秋季腹泻怎么办"
```

**时长档位系统：**

| 档位 | 时长 | 分镜数 | 结构 | 文案字数 | 适用场景 |
|------|------|--------|------|----------|----------|
| ⚡极速版 | 15-30秒 | 3-5个 | 钩子(5s)→单点价值(15-20s)→收尾(5s) | 80-150字 | 单一辟谣、节日祝福 |
| 🔥标准版 | 30-60秒 | 5-7个 | 钩子(5-8s)→2-3个要点(20-35s)→收尾(8-10s) | 150-300字 | 单问题科普、技巧合集 |
| 🎯完整版 | 60-120秒 | 7-12个 | 三层完整骨架 | 300-500字 | 完整对比、原理讲解 |
| 📚深度版 | 120-180秒 | 12-18个 | 三层骨架+分段递进 | 500-800字 | 复杂多维度、分龄段科普 |

**节奏自动计算：**
- 节奏刺激点数量 = floor(总时长 / 18)
- 维度切换次数 = floor(总时长 / 20)

**时长校验：**
- 建议范围：15秒～180秒
- 超出范围提示用户调整到范围内

**选题范围：不限。** 用户可自由提出任何与儿童/母婴/家庭健康科普相关的选题。脚本框架会按选题类型自动适配对应的叙事策略（急救类→危机钩子+步骤拆解，习惯培养类→场景代入+互动强化，科普类→认知冲突+知识拆解）。

### 2.2 用户输入不足时的处理

如果用户只给了时长和选题，没有提供足够的具体方向，主动询问以下信息来完善脚本：

- **目标受众年龄** — 幼儿（2-4岁）/ 学龄前（4-6岁）/ 学龄儿童（6-10岁）？不同年龄段的叙事复杂度、语速、知识深度需要相应调整
- **核心知识点** — 这个视频最想让观众记住的一个核心信息是什么？如有多个知识点，按优先级排列（脚本核心只讲1-2个，其余可做延伸/花字补充）
- **场景偏好** — 更倾向哪种场景风格？
  - ① **家庭生活场景**（真实感家居，代入感强）
  - ② **纯色背景科普场景**（极简背景，专注知识点讲解，AI最稳）
  - ③ **科技感科普场景**（柔和渐变+发光元素，适合抽象概念可视化）
  - ④ **卡通体内冒险**（人体内部拟人化，核心视觉爆点）
  - ⑤ **诊室/医疗场景**（专业感+亲和力）
  - ⑥ **户外场景**（阳光自然，适合运动/习惯类）
  - ⑦ **混合切换**（默认选项，按叙事需要多场景交替）
- **互动形式** — 有没有希望强化的互动环节？如"结尾设置有奖问答"、"引导家长和孩子一起做"等
- **角色使用** — 是否使用丁桂儿IP形象？如果否，是否有其他角色偏好？

如果用户表示"你看着办"或"随便"，则按以下默认策略自动选择：
- 受众默认：**学龄前（4-6岁）** — 叙事节奏适中，知识点拆分最细，兼顾趣味性和教育性
- 场景默认：**家庭生活场景为主** + 纯色背景科普场景做知识点拆解 + 卡通体内冒险段点缀（混合切换，保持视觉丰富度）
- 互动默认：**知识点提问 + 收藏引导** — 最常见且转化效果稳定的互动组合
- 角色默认：**丁桂儿IP** + 辅助小朋友角色

## 三、核心故事结构（爆款四段式）

每个脚本严格遵循以下叙事节奏，对应科普短视频已验证的完播率结构：

### 第1段：钩子抓取（占15%时长）
**目标：0-3秒内强制捕获注意力**

| 钩子类型 | 适用选题 | 话术示例 |
|---------|---------|---------|
| **场景危机型** ⭐优先 | 急救/意外类 | "宝宝呛住了！别慌，这样做！" |
| **认知冲突型** | 辟谣/误区类 | "退烧贴真的能退烧吗？90%的家长都做错了" |
| **痛点提问型** | 日常养育类 | "你家孩子是不是一到换季就中招？" |
| **身份标签型** | 习惯培养类 | "所有宝妈都在犯的同一个洗手误区" |

### 第2段：场景代入（占25%时长）
**目标：建立代入感，抛出具体问题**

- 起承转合中的「起」——日常生活场景展开
- 丁桂儿观察到小朋友出现状况（表情变化：好奇→关注→认真，看向画面中其他角色）
- 旁白抛出问题：用第二人称"你""宝宝"拉近距离
- 综艺花字配合弹出问题核心词

### 第3段：知识拆解（占45%时长）
**目标：分步讲解，知识点钩子每15秒一个**

- 起承转合中的「承+转」——拆解→核心知识点爆发
- **必杀技：拟人化类比法** — "把抽象知识藏进故事里"
  - 渗透压 → "水往浓度高的地方跑"
  - 免疫细胞 → "身体里的小卫士在打仗"
  - 病毒传播 → "坏蛋悄悄溜进来了"
  - **⭐ 身体内部大冒险** — 画面从现实场景缩小/穿越进入卡通版人体内部，丁桂儿化身"小导游"带领观众漫游器官世界（如脾胃像"食物搅拌工厂"、肠道像"弯弯曲曲的滑梯管道"、肺泡像"一串串小气球"）
- 画面配合：每7-10秒切换一次镜头/信息点
- 操作步骤不超过3个关键点（人脑短期记忆上限）
- 每个步骤配"动词+结果"短句，避免"首先、其次"等逻辑词
- 关键数字用可视化对比呈现（不要只说"5cm"，要画一把尺子对比）

### 第4段：互动收尾（占15%时长）
**目标：总结要点 + 互动引导 + 情绪共鸣**

- 起承转合中的「结」——知识点总结回归生活
- 丁桂儿面向镜头的互动动作（点赞/握拳/比心）
- **互动埋点必选其一：**
  - 知识点提问："记住了吗？评论区告诉我第一步该做什么"
  - 转发引导："转给家里带娃的人看看"
  - 收藏引导："收藏起来，关键时候用得上"
  - 下期预告："下期教你怎么区分感冒和流感"
- 综艺总结花字框弹出 + 丁桂儿小课堂logo

### 3.6 钩子公式选择器 — 选type即出公式

每种钩子类型配标准化话术公式+对应画面模板。输出脚本时直接选择对应的type并按公式生成。

| 类型 | 话术公式 | 画面模板 |
|------|---------|---------|
| **场景危机型** ⭐优先 | `[紧急动词]+[对象]！[正确动作]，否则[后果]` | 前2秒丁桂儿震惊表情+红色警示花字"⚠️[危机词]" |
| **认知冲突型** | `[常识做法]真的对吗？[数字]%的人[错了]` | 丁桂儿歪头疑惑+头顶问号气泡+红❌划掉错误做法 |
| **痛点提问型** | `你家孩子一[场景]就[症状]？其实[真正原因]` | 丁桂儿模仿症状表情+场景代入+症状图标悬浮 |
| **身份标签型** | `所有[身份]还在信[误区]？[正确做法]才是对的` | 丁桂儿严肃摇头+×掉常见做法+✓弹出正确做法 |

**示例（场景危机型）：**
- 话术→"宝宝呛住了！别慌，海姆立克这样做！"
- 画面→丁桂儿瞪大眼睛+双手做出急救姿势+红底白字"⚠️窒息急救"

**示例（认知冲突型）：**
- 话术→"退烧贴真的能退烧吗？90%的家长都做错了"
- 画面→丁桂儿指着退烧贴摇头+头顶"？"气泡+划掉退烧贴的❌动画

### 3.7 双钩组合规则

> 适用于复杂度较高（≥60秒）的脚本，简单脚本保持单钩。

1. **允许组合**：仅「认知冲突→场景危机」这一种组合顺序有效（先制造"啊？"再加压紧迫感）
2. **禁止组合**：同类型双钩不可连打（不能两个场景危机连用）
3. **默认策略**：除非用户明确要求双钩，默认单钩起步。脚本地复杂度≤30秒不用双钩
4. **一句话**：一个选题最多两个钩子，且必须不同type，顺序不可互换

### 3.8 风格禁区（触碰即废）

以下红线碰都不要碰：

| 禁区 | 说明 |
|------|------|
| ❌ 专业术语不做比喻转化 | 每个专业概念必须用生活化比喻翻译，不能直接扔给观众 |
| ❌ 直白产品推销台词 | 即使启用产品植入段，也永远用"可以试试"代替"快买" |
| ❌ 快节奏跳切、爆炸特效 | 儿童向科普转场全部柔和，禁止眩光/爆炸/闪白 |
| ❌ 文字遮挡角色主体 | 花字/字幕永远避开角色脸部和关键动作区域 |
| ❌ 刺眼高饱和配色 | 全程马卡龙低饱和配色，禁用荧光色/霓虹色 |
| ❌ 开场无钩子 | 前3秒必须有视觉冲突点，不允许"大家好我是XX"式开场 |
| ❌ 全片无情绪起伏 | 情绪曲线必须设计：平静→紧张→豁然，全程平的等于白做 |
| ❌ 角色台词说教书面化 | 丁桂儿说话像背课文→观众秒滑走 |
| ❌ 信息过密 | 一个知识点讲透再讲下一个，不要一个镜头塞三个信息点 |
| ❌ 动画类超20秒无新刺激点 | 每15-20秒必须刷新视觉/信息刺激，否则完播率崩 |
| ❌ AI画面包含文字/字幕 | 画面提示词禁止写入文字——所有文字由后期添加（详见§7.6.2） |
| ❌ 场景图风格不统一、光影跳变 | 同一视频所有场景图保持一致的画风、色调、光影 |

## 四、互动设计体系

### 4.1 分镜内互动埋点

每个分镜至少设计一个互动触发点，按优先级排序：

| 优先级 | 互动类型 | 实现方式 | 目标 |
|--------|---------|---------|------|
| ⭐⭐⭐ | 悬念钩子 | 分镜末尾留问号 | 拉高整体完播率 |
| ⭐⭐ | 认知冲突 | 打破常识制造"啊？"时刻 | 激发评论 |
| ⭐⭐ | 场景提问 | "你家宝宝也这样吗？" | 引发共鸣 |
| ⭐ | 数字悬念 | "这个动作做错的人超过80%" | 制造紧迫感 |

### 4.2 旁白互动话术库

| 场景 | 话术模板 |
|------|---------|
| 开头钩子 | "宝宝XX的时候，[常见错误做法]？**错！**" |
| 知识讲解 | "记住三个字——**[关键词]**，一[动作1]，二[动作2]，三[动作3]" |
| 紧急性强调 | "黄金XX分钟，**别做错这一步**" |
| 误区纠正 | "[常见做法]不但是错的，还可能[危害]！正确的做法是——" |
| 结尾互动 | "学会了吗？**截图保存**，转发给家人一起看～" |

### 4.3 高传播台词设计模块

在每篇脚本中至少插入1句"可截图传播"的台词——让观众看完想截下来发朋友圈/发群的台词。

**核心原则：** 台词要有「被单拎出来也能火」的独立传播力。它不一定需要画面背景，光文字本身就能让人get到情绪。

**台词类型库：**

| 类型 | 适用场景 | 示例 |
|------|---------|------|
| 自嘲共鸣型 | 宝妈日常共鸣 | "当妈之后才知道，累是随时随地的事" |
| 夸张吐槽型 | 知识拆解段末尾缓解密度 | "如履薄冰啊——每个当妈的都是薄冰哥" |
| 角色反差型 | 丁桂儿突然冒出大人话 | "你们人类啊……（摇头）" |
| 神转折型 | 从科普突然拐弯 | "本来以为是小问题，结果——" |
| 冷面萌吐槽型 | 丁桂儿被问题气到 | "这话我没法接……但我可以教你怎么做" |

**插入方法：**
- 知识拆解段末尾插1句吐槽（缓解信息密度）
- 互动收尾段插1句反差（强化记忆点）
- 不强行插入——如选题不适合（纯急救类），跳过

**可截图时刻绑定（关键）：**
每个高传播台词分镜必须标注「截图时刻」，确保这句台词对应的画面截图出去本身是一张表情包：

> **截图时刻标准：**
> - 分镜最后2秒为「可截图帧」
> - 可截图帧画面要求：
>   - 丁桂儿面部占据画面30-50%
>   - 表情处于台词情绪峰值（惊讶/无语/得意/眯眼笑）
>   - 背景简洁，无遮挡面部元素
>   - 画面上无其他文字元素干扰

### 4.4 双层台词分层管理体系

丁桂儿相关台词分两层，互不干扰但并行输出：

| 层级 | 占比 | 面向受众 | 风格 | 梗来源 | 输出位置 |
|------|------|---------|------|--------|---------|
| **儿童层 🧒** | 70% | 2-6岁儿童观众 | 简单句、拟声词、重复句式、动物比喻 | ❌不用网络梗，用儿歌/童谣节奏 | 丁桂儿口中台词 |
| **家长层 👩** | 30% | 观看的宝妈/家长 | 制造转发欲、评论区共鸣、吐槽 | ✅网络梗库（详见§4.5） | 旁白/花字/丁桂儿OS低语 |

**不割裂的方式：**
- 画面里丁桂儿正常说儿童层台词（小朋友听得懂）
- 花字/旁白同时弹出家长层梗（字幕一飘而过，家长看到了忍不住截图）
- 如果家长层台词通过丁桂儿说出来——用"OS内心独白"形式，音量调到30%，画面中丁桂儿表情留一个"有苦说不出"的微妙变化

**示例：**
```
画面：丁桂儿教小朋友洗手步骤
丁桂儿口中（儿童层）："一搓搓，二揉揉，三冲冲——洗好啦！"
花字（家长层）："当妈才知道什么叫随地大小累" ← 家长看到会心一笑截图
```

### 4.5 台词/梗选取逻辑

从网络梗库（`~/.claude/memory/internet-slang.md`，约240条）中选取台词时，按以下优先级进行：

1. **选题关键词匹配优先** — 选题含"咳嗽/腹泻/发烧/便秘"→优先从梗库中检索含对应关键词的梗
2. **场景情绪匹配** — 情感节点选题→优先用共鸣类梗；科普选题→优先用吐槽类梗
3. **频率控制** — 同一条视频最多用**2个网络梗**，剩下一句用冷面萌语录式原创台词
4. **轮换机制** — 连续3次输出不重复使用同一句梗（有记录则跳过，选次优匹配）
5. **边界** — 成人话题/抽象谐音梗/纯文字冷笑话梗不选入儿童科普脚本

**梗库不靠每次联网更新**：以现有梗库为默认语料。用户每隔1-2个月扔几个新梗来，手动更新梗库文件。只有在选题需要当季热梗时（如开学季/春节），单独联网搜索一次即可。

## 五、科普节奏控制

### 5.1 节奏参数表

| 参数 | 标准 | 说明 |
|------|------|------|
| 镜头切换频率 | 每5-8秒 | 适配儿童注意力跨度，较成人短视频更快 |
| 知识点钩子 | 每12-15秒一个 | 防止信息过载+注意力流失 |
| 旁白语速 | 中偏慢，120-140字/分钟 | 像讲故事一样，一句一句说清楚 |
| 关键信息重复 | 每集重复2-3次核心操作 | 强化记忆 |
| 步骤数量 | ≤3个关键步骤 | 人脑短期记忆上限 |
| 呼吸点 | 每12-20秒插入一处 | 短暂留白/画面停顿/自然音效，让儿童消化信息 |
| 互动频率 | 每30-45秒一次 | 互动时长不超过8秒 |
| 画面转场 | 每组分镜标注转场方式 | 硬切/叠化/缩放/动势/模糊转场 |

### 5.2 节奏设计原则

1. **每5-8秒触发一次视觉更新**：景别切换、角色动作变化、花字弹出、场景变换
2. **每12-15秒触发一次"啊哈时刻"**：核心知识点露出、反转、数据冲击
3. **呼吸点设计**：每12-20秒插入一处"信息消化间隙"——短暂留白、自然音效、画面缓停，让儿童有喘息余地，避免信息轰炸
4. **旁白留白控制**：说一句关键信息→停顿1-2秒→让画面和花字消化信息
5. **情绪曲线**：平静/好奇（开头）→紧张/疑惑（中段）→豁然/安心（结尾）
6. **声音先导（J-Cut原则）**：下一个镜头的声音提前0.5-1秒出现，画面稍后再切——用声音引导观众视线，减少镜头切换的突兀感
7. **声音延续（L-Cut原则）**：画面切走后，上一个镜头的旁白/音效延续0.5-1.5秒——保持叙事连贯，过渡更自然
8. **音效卡点**：关键知识点露出的瞬间，用"叮/噔"等短促音效对齐画面动作，形成视听同步冲击

### 5.3 景别交替规律

**核心原则：避免连续同景别，形成循环视觉节奏**

景别按以下循环模式排布，每3-4个镜头完成一次周期：

| 景别 | 科普用途 | 占比建议 | 使用场景 |
|------|---------|---------|---------|
| **远景/全景** | 建立空间关系 | 10% | 展示生活场景全貌、体内环境全貌 |
| **中景** | 主干知识讲解 | 35% | 丁桂儿讲解、与其他角色互动、操作演示 |
| **近景** | 表情+操作细节 | 25% | 角色情绪变化、手部动作特写 |
| **特写/大特写** | 关键信息强调 | **30%** ⭐ | 食物/道具细节、器官动画、症状可视化 |

**推荐循环模式：**
```
全景定位（告知观众在哪里）
  → 中景叙述（丁桂儿开讲）
    → 近景/特写强调（关键知识点可视化）
      → 中景回归（回到丁桂儿，进入下一个知识循环）
```

> **特写是儿童科普的灵魂镜头**——显微镜下的细胞、气泡炸裂、食物在胃里的搅拌过程，这些肉眼看不清的细节用特写呈现，是完播率的核心驱动力。

## 六、拍摄手法与视觉语言体系

### 6.1 八大核心运镜手法（科普场景适配版）

每个分镜的"运镜手法"字段可从以下库中选用，根据科普场景匹配最合适的运镜方式：

| # | 运镜手法 | 科普应用场景 | 操作说明 |
|---|---------|-------------|---------|
| 1 | **推镜** (Push In) | 从场景全景推至角色/道具特写，引导观众聚焦核心 | 缓慢推入营造"观察感"，快速推入制造"紧张感" |
| 2 | **拉镜** (Pull Out) | 从特写拉出揭示全貌，制造"原来如此"的豁然开朗感 | 如从一片食物特写拉出到整个餐桌场景 |
| 3 | **跟镜/移镜** (Tracking) | 跟随角色运动轨迹，沉浸式展示操作过程 | 丁桂儿边走边讲解，镜头平行跟随；或追踪食物在体内的运动路线 |
| 4 | **环绕运镜** (Orbit) | 360°展示物体全貌，适合道具/模型展示 | 缓慢环绕展示食物模型、器官结构、药膳碗等；速度要慢且稳定 |
| 5 | **俯仰运镜** (Tilt) | 上下摆动镜头，强调物体的大小对比或高低关系 | 从小朋友仰视宝妈→上摇到宝妈俯视表情，强化身高差 |
| 6 | **第一人称视角** (POV) ⭐ | 模拟观众视角，强代入感——"好像自己在操作" | 模拟小朋友看自己肚子的视角、模拟动手做推拿的第一人称手部视角 |
| 7 | **升降运镜** (Crane) | 垂直方向上下移动，展示层次或高低差 | 从地面低角度升起至平视，或从高处俯瞰降下至角色水平 |
| 8 | **甩镜/快速横移** (Whip Pan) | 快速从一个物体甩到另一个，表现思维跳跃/对比 | 从"错误食物"快速甩到"正确食物"，配合"错！"旁白 |

**运镜选择优先级：**
- 讲解段 → 固定机位 / 轻微推镜（信息稳定，不干扰认知）
- 操作演示段 → 跟镜 / POV（沉浸感最强）
- 体内冒险段 → 推镜进入 + 跟随运动（穿越感）
- 对比/禁忌段 → 甩镜 / 快速横移（节奏冲击）
- 情感共鸣段 → 缓慢拉镜 / 环绕（温馨氛围）

### 6.2 构图策略

| 构图方式 | 科普用途 | 说明 |
|---------|---------|------|
| **中心构图** | 主体讲解、关键信息展示 | 丁桂儿正面面对镜头讲解知识，信息直接 |
| **三分法构图** | 角色+道具/场景并置 | 丁桂儿居左/右1/3，指向画面中的道具或另一角色 |
| **斜角构图** | 制造紧张/冲突感 | 用于误区警示段，稍微倾斜画面增加不安感 |
| **俯拍构图** ⭐ | 操作演示（推拿/按摩/食物摆盘） | 从正上方垂直拍摄操作台面，清晰展示手部动作全貌 |
| **仰拍构图** | 强化身高差/权威感 | 从低角度拍摄宝妈/医生，配合小朋友仰视视角 |
| **对称构图** | 对比展示（对与错、吃前vs吃后） | 画面左右分屏，左边错误做法+右边正确做法 |
| **引导线构图** | 引导视线指向关键元素 | 利用餐桌边缘/地板线条/箭头道具引导视线指向核心道具 |

### 6.3 剪辑声画技巧

**J-Cut（声音先行）：**
- **做法**：下一个镜头的旁白/音效提前0.5-1秒出现，画面稍后切入
- **效果**：用声音引导观众视线转向，减少镜头切换的突兀感
- **科普应用**：屏幕还停留在"食物"画面时，下一段讲解声先入→观众注意力被声音提前带走→画面自然切换到讲解场景
- **分镜标注**：在分镜"转场"字段末尾添加 `[J-Cut：旁白提前0.8s入]`

**L-Cut（声音延续）：**
- **做法**：画面切走后，上一镜头的旁白/音效延续0.5-1.5秒
- **效果**：保持叙事连贯，过渡自然，避免"切一下断一下"的碎片感
- **科普应用**：画面已切到体内场景，但丁桂儿的旁白"让我们一起来看看——"延续进来，让穿越转场更柔和
- **分镜标注**：在分镜"转场"字段末尾添加 `[L-Cut：旁白延续+0.8s]`

### 6.4 音效设计体系

| 音效类型 | 音效示例 | 触发场景 | 作用 |
|---------|---------|---------|------|
| **知识点强调** | 叮！噔！叮咚！ | 关键数据/知识点弹出的瞬间 | 标记"记住这个" |
| **警示** | 警报BEEP！噔噔！ | 错误做法/禁忌内容出现时 | 引起警觉，配合红框花字 |
| **转场过渡** | 咻——、嗖、噗 | 画面切换/场景穿越 | 平滑过渡，制造动态感 |
| **拟声** | 咕咚（喝水）、咕噜噜（肚子叫）、啊啾（喷嚏） | 配合角色动作 | 增强临场感和趣味性 |
| **情绪** | 欸？（疑惑）、哎呀！（惊讶）、嗯~（思考） | 丁桂儿表情变化时 | 强化情绪传递，适配低龄观众 |
| **节奏** | 滴答声、轻快鼓点 | 紧迫感段落/倒计时 | 制造节奏紧张感 |
| **旁白留白音效** | 温柔叮咚、轻快噔噔 | 呼吸点/停顿处 | 填补信息间隙，保持听觉连贯 |

**音效使用原则：**
- 知识点强调音效必须**对齐画面动作的峰值帧**（如花字弹出一瞬间）
- 拟声音效比背景音乐更重要——儿童对具体声音比抽象旋律更敏感
- BGM音量控制在旁白的**60%以下**，避免喧宾夺主
- 同一条视频中，同类型音效应避免连续使用超过3次

### 6.5 花字统一性铁律（单条视频强制执行）

> 以下规则在每篇脚本输出时必须遵守，从第一个分镜到最后一个分镜全程锁定。

**字体选择：**
- **主字体推荐**：综艺体 / 胖胖体 / 手写萌体——必须有**综艺感**（圆润、粗体、有描边/阴影、有节奏冲击力）
- **禁止使用**：宋体、黑体、楷体、衬线体、极细体（不够综艺，太正经/太像教科书）

**7条硬规范：**

| # | 规则 | 说明 |
|---|------|------|
| 1 | **主字体锁定** | 一条视频只用一个主字体，跨分镜不换字型 |
| 2 | **配色锁定** | 全片花字用同一套配色方案，最多2色。推荐：白字+红描边 / 黄字+黑描边 / 白字+蓝圆角底 |
| 3 | **字号层级固定** | 标题/警示/情绪爆发=字号1（最大，占画面宽度60-80%）；知识点标注=字号2（带色块底或圆角框）；旁白字幕=字号3（底部居中，半透明底） |
| 4 | **动画类型一致** | 同类型文字用同一入场动画。综艺推荐动画：弹入+轻弹跳 / 缩放带旋转 / 从底部滑入带弧度 |
| 5 | **位置避让一致** | 花字不遮挡丁桂儿面部和关键道具——具体位置见下条动态规则 |
| 6 | **不得跨分镜切换样式** | 分镜1用红框白字→分镜5不能突然改蓝框黄字。每个分镜标注"此段花字样式参考分镜X" |
| 7 | **综艺感增强技巧** | 关键数字/词用「增大+换色+弹跳入场」三连；警示词加⚠️+红框+抖动动画三件套；情绪台词加emoji/星标/波浪线点缀 |

**花字位置动态适配规则（非固定位置）：**

花字位置不由"左上/右上"硬性指定，而是根据**画面构图留白**动态决定：

- 丁桂儿在画面左侧→花字放右上/中右
- 丁桂儿在画面右侧→花字放左上/中左
- 丁桂儿居中→花字放下方1/3区域（半透明底）
- 对称构图→花字放下方

**硬性底线：**
- 花字不得覆盖丁桂儿面部（≥50%面部可见）
- 花字不得遮挡关键道具/操作手部
- 画面中央60%区域为"主角保护区"，花字默认不侵入
- 一个分镜中花字最多出现2条，单条≤6字

## 七、输出格式标准

### 【氛围与画质】
风格核心: Q版2D卡通、亲子综艺感、明亮治愈、儿童向科普动画、流畅平涂渲染质感，杜绝生硬3D建模感，杜绝角色脸崩、画风割裂。整体氛围兼具科普严谨性与紧急情况下的冷静指导感（急救类）或轻松趣味感（习惯类），适配亲子家庭观看。
视觉基调: 9:16竖屏画幅，分辨率1080×1920，高清卡通画质，线条圆润柔和，画面干净通透，主体人物突出。
色彩与影调: 以明黄、天蓝为主色调，严格匹配丁桂儿IP形象色系，采用明亮柔和的马卡龙配色，整体营造温馨但专注的家庭科普氛围；光线均匀柔和，无强烈明暗反差；搭配急救主题的综艺花字、警示标签、步骤高亮框等装饰元素，强化教学感。**体内微观场景**另行采用半透明渐变光效+柔光晕染风格，器官用粉嫩暖色系（淡粉/浅橙/米黄），营造"可爱工坊"感而非医学真实感。
字幕呈现: 搭配综艺特效花字字幕，字体圆润可爱，关键操作步骤用醒目红/橙色高亮标注，字幕动效配合节奏停顿清晰呈现，避免信息过载。考虑到80%用户静音观看，**字幕必须完整呈现所有关键信息**，重点词标红/加粗/放大。

**视频形式配色表：**

| 视频形式 | 主色调 |
|----------|--------|
| 纯科普动画 | 蓝绿色科技感，低饱和 |
| 实拍+动画混剪 | 暖黄居家柔光 |
| 情感节日类 | 马卡龙低饱和 |
| 干货技巧类 | 浅清冷科普调 |

字体规范：圆润柔和无衬线，重点加粗高亮，文字始终避开角色主体。

**文案规范（输出必须遵守）：**
- 单句≤15字，一个镜头1-2句
- 口语化，去专业术语，融入情绪语气词
- **标签双矩阵**：每次输出末尾附标签矩阵（公域2个 + 垂直4个 = 共6个，详见§7.7）
- 花字与旁白文字百分之百同步，不允许旁白说A花字显示B

**节奏补充规范：**
- 整体"紧-松-紧-松"交替，开场前5秒全片最高刺激点
- 动画类每15-20秒一个新刺激点，实拍+动画混剪类每20秒切换一个维度
- BGM不抢核心信息，音量控制在旁白的60%以下

### 【基础设定】
核心角色-丁桂儿: Q版卡通幼儿形象，定位为**小讲解员/小主持人**（类似儿童节目中的知识引导者），头顶一撮标志性黑色刘海，圆脸蛋、大眼睛，表情活泼生动、富有感染力；身穿明黄色短袖T恤（胸前印有红色"丁桂"字样）、蓝色休闲短裤、蓝白拼色运动鞋；身形比例圆润可爱。全程脸型、发型、服装、颜色、整体比例保持完全一致，每一帧形象统一无偏差。
丁桂儿的核心叙事功能：
- **讲解员角色**：直面镜头讲解知识，用手势引导观众注意画面中的关键内容，指向案例中的其他角色或场景元素
- **互动桥梁**：与观众（镜头外）建立直接对话感，通过提问、眼神交流、动作引导拉近距离
- **不取代案例角色**：丁桂儿不充当"生病/出状况"的角色——由出场的小朋友饰演具体案例，丁桂儿以旁观+讲解者身份出现
- **与辅助角色互动**：可与其他小朋友互动（演示动作、检查步骤、一起做操等），但始终保持"我知道怎么做，我来教你"的引导者姿态
丁桂儿表情系统（按场景调用）:
- 开心/得意：眯眼笑，脸蛋泛红晕

### 场景选择策略（按选题类型推荐场景组合）

**场景切换节奏：**
- 一集中场景切换不超过 **3种类型**，避免画面割裂感过强
- 纯色背景和科技感场景最适合作为"知识讲解段"的稳定锚点，适合长时间讲解不疲劳
- 生活化场景和体内冒险场景交替时，必须使用**缩放转场/穿越特效**过渡，不可硬切
- 纯色背景场景之间切换颜色时（如白色→浅蓝），用叠化过渡，0.5秒即可

**即梦AI适配提示：**
- **纯色背景场景**即梦还原度极高，提示词前置"纯色背景、干净简洁、丁桂儿居中"，加入"漂浮的卡通[器官/食物/细菌]图标作为辅助元素"
- **科技感场景**提示词中用"柔和渐变、发光线条、全息投影、Q版科技感"，避免"赛博朋克/霓虹/暗黑"等过于硬核的关键词
- 生活化场景需在提示词中明确空间（客厅/卧室/幼儿园教室）+ 光源方向 + 时间感（早晨/午后/傍晚）
声音: 无需背景音乐，仅保留活泼亲切的女性宝妈音旁白 + 轻量卡通拟声音效；旁白语速适中偏慢（120-140字/分钟），吐字清晰，关键步骤加重语气；急救类选题可增加心跳声/警报声作为开头钩子的紧迫感音效。

**声音统一性专项规范（跨分镜锁定，防止音色漂移）：**

**【旁白音色锁定参数（TTS/配音指定）】**
- 音色类型：温柔亲切宝妈音（温暖中音区，带轻微笑意），严禁使用播音腔/严肃新闻腔/儿童尖细音
- 语速基线：120-130字/分钟，全程恒定不变。知识拆解段维持基线，警示段可短暂提速至140字/分钟但不超过3秒后必须回归
- 音调区间：中音区为主（220Hz-280Hz），知识点加重处可升至高音区（300Hz）但不超过3秒
- 情绪基准：全程保持"关心但冷静"的口吻基调——开头关切、中间理性讲解、结尾温和鼓励，不出现大幅情绪波动
- 咬字风格：字正腔圆但不僵硬，句尾不拖音。儿童科普专用TTS参数：温柔度80%、清晰度95%、情感度60%
- 跨段锁定：每个分镜独立生成时必须使用完全一致的TTS预设参数（voice_id/pitch/speed/timbre），不可手动微调单段

**【丁桂儿声音表现规范（对话/拟声/情绪发音）】**
- 声线定位：明亮活泼的男童音（非女声伪男童），清脆有穿透力但不过度尖锐
- 情绪发音库（按分镜锁定调用，不得跨段漂移）：
  - 惊讶/不适时：「啊啾！」短促清脆喷嚏声 + 轻声「唔…」表示难受——分镜1专用
  - 认真讲解时：食指竖起「记住哦~」语气上扬带笑意——讲解段专用（分镜2/分镜4等）
  - 活泼类比时：「大军压境！」加重语气+略带夸张——拟人化知识段专用（分镜3等）
  - 开心收尾时：眯眼笑+「记住啦？」带问号上扬——收尾段专用（末镜）
- 禁用项：全程禁止出现机械TTS合成感、气息不稳、音量忽大忽小、同一句尾音忽降忽升

**【AI生成工具声音一致性操作规范（六步锁定法）】**
1. **先导定音**：在所有分镜生成前，先用收尾镜（标准口吻+开心语调）跑一次测试，导出voice_id锁定
2. **参数固化**：将确定的voice_id、pitch、speed、timbre记录为固定参数模板，每个分镜生成时直接套用
3. **批量生成**：不要逐段生成逐段调音。所有分镜以完全一致的TTS参数批量输出旁白音轨
4. **预混验证**：将各分镜导出的音轨在剪辑时间线中拼接，连续播放检查是否存在音色突变/语速变化/情绪断裂
5. **补录规则**：若某段必须重新生成，不得手工模仿录制——必须回源工具使用同一voice_id重新渲染
6. **丁桂儿人声分离**：若使用真人配音而非TTS，丁桂儿所有情绪发音（喷嚏、语气词、短句）必须由同一位配音演员在单次录音中一条完成，禁止分日/分次录制

AI生成适配: 本提示词同时适配以下工具——
- **Runway Gen-3**：保留完整视觉描述，增加运动幅度说明
- **Pika Labs**：强化角色一致性描述，每帧独立完整
- **Kling Kool**：增加镜头运动关键词（push in/pan/tilt）
- **Sora**：增加物理交互细节描述（手部动作、物体受力反馈）

#### 7.8.1 封面类型选择器（按选题类型推荐）

#### 7.8.2 封面CTR三要素

1. **画面主体**：丁桂儿面部始终占据画面30-50%，表情传递情绪
2. **标题字**：≤12字，前4字含核心关键词，字号大于主体元素
3. **配色**：主色1个+辅色1个+点缀色1个，保持低饱和马卡龙调性

> ⚠️ **数据说明：** 以上CTR优化方向基于儿童内容赛道通用规律及拆解文档中的画面结构分析，非本账号实测数据。建议发布后监测点击率，进行A/B测试验证。

#### 7.8.3 封面设计红线

- ❌ 丁桂儿面部被文字遮挡
- ❌ 颜色超过3种
- ❌ 纯文字无角色
- ❌ 一张封面用遍所有选题（必须按类型差异化）
- ❌ 封面中出现AI生成的文字（封面标题由后期添加，非AI生成）

## 十一、口播文字输出规范

每次输出分镜脚本时，必须同步输出一份**完整逐字口播稿**和**正确性校验表**。

### 11.1 口播文字输出要求

口播稿与分镜脚本一一对应，按时间分段输出：

```
**▶ X-Y秒**
> [完整旁白台词，精确到每个字]

**▶ Y-Z秒**
> [...]
```

**口播稿规范：**
- 语速控制在 **120-140字/分钟**，根据总时长倒推总字数上限
  - 30秒脚本：口播总字数 ≤70字
  - 40秒脚本：口播总字数 ≤95字
  - 60秒脚本：口播总字数 ≤140字
  - 90秒脚本：口播总字数 ≤210字
- 口播中的关键数据/操作/警示词用 **加粗** 标注（对应分镜中的红框花字）
- 每条口播开头标注对应的时间区间，方便剪辑直接对轨
- 每个知识点的口播必须与画面内容完全同步，不得超前或滞后于画面信息量
- 钩子段结尾故意设计"悬念断句"（如"第一件事——"），配合分镜的悬念设计

### 11.2 正确性校验表

口播稿末尾必须附加校验表，用文字排版（不用表格），逐条核验关键医学/科普信息的来源依据：

```
**✅ 正确性校验**

---
口播内容：[关键数据/建议]
依据：[依据来源，如临床指南/专家共识/教材]

---
口播内容：[关键数据/建议]
依据：[依据来源，如临床指南/专家共识/教材]
---

**校验规则：**
1. 每个转述给家长的操作指令（剂量、温度、时间、步骤）必须附依据
2. 依据优先引用：中国儿科/儿保相关临床指南 > 权威教材 > 专家共识
3. 无法核实的信息不得出现（宁可不说不乱说）
4. 争议性护理方法（如物理降温、止咳药使用）必须标注共识现状
5. 明显违反安全常识的"民间偏方"必须在表中标注"错误做法/不推荐"
```

### 11.3 文字正确性校验强化规则（防AI幻觉/术语混淆）

在完成校验表的基础上，额外逐条核查以下细节：

1. **数字零误差**：每一条旁白中的数字（温度/天数/百分比）必须在输出前经校验表逐一比对，零误差容忍
2. **术语使用规范**：「流感」不得简写为「感」；「普通感冒」不得简称为「感冒」以免混淆；「抗生素」不得误写为「消炎药」
3. **口诀反向验证**：口诀类内容输出后反向验证是否覆盖全部核心知识点——口诀遗漏则补口诀
4. **花字-旁白同步**：花字与旁白的文字必须百分之百同步——旁白念什么花字写什么，不允许出现旁白说A、花字显示B的分裂情况
5. **跨语言格式统一**：中英混用时英文大小写一致（如39℃不得写成39°C）
6. **禁止AI幻觉**：所有医学数据必须有据可查，校验表中依据来源必须精确到书名/指南名+年份，禁止笼统写"权威资料"

## 十二、输出自检清单

输出前逐项校验，不达标则修正：

- [ ] **钩子有效性**：前3秒是否包含明确的冲突/危机/反常识钩子？钩子类型是否已选择并匹配公式？
- [ ] **双钩合规**：如果使用双钩，顺序是否为"认知冲突→场景危机"？不同type？
- [ ] **节奏合规**：每个分镜时长是否在7-10秒区间内？是否每15秒有知识点钩子？
- [ ] **IP一致性**：丁桂儿的脸型、发型、服装、颜色、比例全程统一，无矛盾？
- [ ] **拟人化角色合规**：如有启用拟人化角色，是否≤3个？出场规则是否匹配选题？
- [ ] **选题匹配确认**：内容是否严格对应所选选题？赛道匹配结果是否标注？
- [ ] **科普准确**：医学/健康内容是否经核实？无危险操作建议？
- [ ] **互动埋点**：脚本中是否至少包含1条互动引导话术？
- [ ] **总时长对齐**：所有分镜时长累加是否等于输入总时长？
- [ ] **转场标注**：每组分镜间是否标注了转场方式？
- [ ] **三段式完整**：【氛围与画质】【基础设定】【画面内容】一级分类是否齐全？
- [ ] **无声适配**：字幕是否覆盖了所有关键信息点（适配静音观看）？
- [ ] **AI画面不含文字**：所有分镜的画面内容/首帧图中是否确认没有任何中文/英文/多位数文字描述？
- [ ] **后期文字标注完整**：每个分镜是否都标注了「📌 后期文字标注」字段？
- [ ] **花字统一性**：全片花字的字体、配色、字号层级、动画类型是否一致？是否有跨分镜切换样式的违规？
- [ ] **花字位置合规**：花字是否覆盖丁桂儿面部/关键道具/主角保护区？是否根据构图动态适配了位置？
- [ ] **高传播台词**：是否包含≥1句可截图传播的台词？是否标注了截图时刻？
- [ ] **双层台词分层**：如有使用网络梗，是否放在家长层（花字/旁白）而非儿童层（丁桂儿口中）？
- [ ] **口播正确性**：是否附带逐字口播稿及医学依据校验表？每条操作指令是否有据可查？
- [ ] **后期物料包**：是否附剪辑对照表、全片文字标注总表、AI素材清单？
- [ ] **标签矩阵**：是否按§7.7格式输出了公域2个+垂直4个标签？
- [ ] **封面策略**：是否按选题类型推荐了封面类型？
- [ ] **Word文件生成**：是否确认Word文档已保存到 `桌面/工作文件/视频/YYYYMMDD-选题.docx`？
- [ ] **旁白音色一致性**：所有分镜旁白是否使用同一voice_id/同一TTS预设？是否经过拼接预混验证？
- [ ] **丁桂儿声音统一性**：所有情绪发音（喷嚏/语气词/短句）是否由同一人/同一TTS模板输出？无跨段漂移？
- [ ] **文字-语音同步**：花字内容与旁白逐字对应，无旁白说A花字显示B的分裂现象？
- [ ] **术语规范**：流感≠感冒、抗生素≠消炎药，所有医学术语使用是否正确一致？
- [ ] **刺激点密度**：节奏刺激点数量是否 ≥ floor(总时长/18)？是否每15-20秒有新刺激点？
- [ ] **角色一致性方案**：每个角色（尤其是丁桂儿）是否标注了三视图/一致性方案？场景切换时角色形象不漂移？
- [ ] **画面无嵌入文字**：首帧图/场景图的AI提示词中未写入文字/字幕内容？
- [ ] **场景图统一性**：同一视频所有场景图的画风、色调、光影一致，不靠背景跳变制造变化？
- [ ] **产品植入合规**（仅启用时）：前面是否有至少2条免费建议铺垫？有无过渡句承接？画面为卡通化展示而非实物包装？旁白避免硬广话术？
